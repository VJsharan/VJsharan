## Recent Work

**[@vjsharan/causal-loop-agent](https://www.npmjs.com/package/@vjsharan/causal-loop-agent)** — Published AI agent on NPM that audits codebases for security issues, outdated dependencies, and production failures using GitHub API and Gemini 2.5 Pro

**JD-TO-CV** — Resume optimizer with SSE streaming that generates ATS-optimized PDFs in <8s; 6-stage pipeline (validate → extract → analyse → map → render → download)

**DevPulse** — Zero-dependency GitHub analytics dashboard in vanilla JS; <800ms first paint with localStorage caching, personality profiling algorithm measuring 5 metrics from GitHub events

**BloodyQuiz** — Real-time multiplayer quiz platform (WebSocket + Gemini API); supports 30-50 concurrent users, won 1st Prize at VAC Intra Hackathon

**Hackumi** — Hackathon management SaaS with PostgreSQL RLS policies, OAuth, and real-time collaboration features

**Resume Analyzer AI** — NLP-based resume scorer using TF-IDF + cosine similarity across 50 roles and 2,500+ skills in <3s (78-85% accuracy)
